### Current Capacity

- **Requests**: ~1,000 req/s
- **Storage**: ~100 GB

### Bottlenecks Identified
1. Database queries (N+1 problem)
2. No caching layer
3. Single API instance
4. Blockchain sync latency

### Scale Plan
1. **Phase 1**: Add Redis cache
2. **Phase 2**: Read replicas (PostgreSQL)
3. **Phase 3**: Horizontal scaling (API)
4. **Phase 4**: Microservices (optional)
